# pybacting

Python wrapper around [bacting](https://github.com/egonw/bacting).

## Installation

Install from GitHub:

```shell
$ pip install git+https://github.com/cthoyt/pybacting
```

Install in development mode:

```shell
$ git clone https://github.com/cthoyt/pybacting
$ cd pybacting
$ pip install --editable .
```
